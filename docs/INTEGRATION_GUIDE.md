# 🚀 ZenWallet Integration Guide (v1.1)

Welcome to the **ZenWallet Payment Gateway**. This guide will help you integrate high-security, real-time wallet payments into your application.

## 🔐 1. Initiate Payment Request

To start a payment, call the `/external/create-request` endpoint from your **backend**. This creates a secure checkout session.

### Request Body
- **Endpoint**: `POST /external/create-request`
- **Method**: `POST`

```json
{
  "amount": 1500,        // Amount in cents (e.g., 1500 = ₹15.00)
  "referenceId": "ORDER_12345", // Your unique internal Order/Invoice ID
  "callbackUrl": "https://myapp.com/payment-callback" // Where user returns after payment
}
```

> [!NOTE]
> `merchantId` is optional in the body; the system automatically link the request to your account using your **API Key**.

### Success Response
```json
{
  "success": true,
  "data": {
    "paymentUrl": "https://payment-via-zenwallet.vercel.app/checkout?token=92kx...",
    "token": "92kx...",
    "expiresAt": "2026-02-28T..."
  }
}
```

**Action**: Redirect your user's browser to the returned `paymentUrl`.

---

## ✅ 2. Verify Transaction

When the user returns to your site (via `callbackUrl`), verify the payment status using your `referenceId`.

- **Endpoint**: `GET /external/verify-reference`
- **Method**: `GET`

### Query Parameters
| Param | Description |
| :--- | :--- |
| `merchantId` | Your ZenWallet Account ID |
| `referenceId` | The `referenceId` you sent in Step 1 |

### Example Request
`GET https://payment-via-zenwallet.vercel.app/api/external/verify-reference?merchantId=m_123&referenceId=ORDER_12345`

### Response
```json
{
  "received": true,
  "message": "YES: Payment is received",
  "details": {
    "amount": "1500.00",
    "id": "txn_88k...",
    "time": "2026-02-28T..."
  }
}
```

## 💻 Sample Implementation (Node.js)

```javascript
const axios = require('axios');

// 1. Create Checkout Session
async function createPayment() {
    const response = await axios.post('https://payment-via-zenwallet.vercel.app/api/external/create-request', {
        amount: 5000,
        referenceId: 'INV_001',
        callbackUrl: 'https://myshop.com/success'
    }, {
        headers: { 'x-api-key': 'YOUR_SECRET_KEY' }
    });
    
    return response.data.data.paymentUrl; // Redirect user here
}

// 2. Verify on Callback
async function verifyPayment(orderId, merchantId) {
    const response = await axios.get('https://payment-via-zenwallet.vercel.app/api/external/verify-reference', {
        params: { merchantId, referenceId: orderId },
        headers: { 'x-api-key': 'YOUR_SECRET_KEY' }
    });
    
    return response.data.received; // true/false
}
```

## 🛡️ Best Practices
1. **Never** expose your `x-api-key` in the frontend (HTML/Client-side JS).
2. **Always** perform the Verification step on your server before delivering service/goods.
3. **Use** unique `referenceId` values for every transaction attempt.
